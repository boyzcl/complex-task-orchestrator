# 需求共识

## 当前任务

围绕“性行为相关的能力和欲望到底被哪些因素影响”这一问题，完成一份面向广泛成年人群的循证医学综述。

## 目标

产出一份中文综述，尽量全面回答：

- 性欲会受到哪些因素影响
- 性功能/性行为相关能力会受到哪些因素影响
- 这些因素之间的大致逻辑关系是什么
- 哪些结论证据更强，哪些结论证据较弱或仍存在不确定性

## 范围

- 不预设单一特定人群，优先覆盖广泛成年人群
- 如不同人群之间存在重要差异，可分类讨论
- 不预设只谈生理或只谈心理，由证据决定最终纳入范围
- 可以纳入生物学、心理、关系、社会文化、疾病、药物、生活方式等因素
- 不提供改善建议、治疗建议或干预方案，重点是解释影响因素与逻辑

## 证据策略

- 采用循证医学思路
- 优先使用系统综述、Meta 分析、权威指南、权威共识
- 在高等级证据不足时，再下沉到大型观察研究、队列研究或关键机制研究
- 明确区分证据较强、证据中等、证据有限或不确定的结论

## 输出风格

- 正文尽量通俗易懂
- 同时保留证据等级与关键出处
- 最终交付应兼顾可读性与可追溯性

## 验收标准

- 能清晰回答“性欲和性功能/能力受哪些因素影响”
- 结构上能区分主要因素类别及其逻辑关系
- 关键结论可回溯到较高等级证据
- 明确标出证据强弱与不确定性
