# 公开案例总览

本目录中的案例是 `curated public assets`，不是原始 runtime 日志。

它们的作用是：

- 帮陌生用户快速理解这个 skill 适合什么任务
- 展示这套协议在不同任务类型中的实际价值
- 作为公开证据层，而不是私有运行样本层

## 当前案例

- [案例 1：研究 / 综述任务](case-01-research-review.md)
- [案例 2：skill 自举创建任务](case-02-skill-bootstrap.md)
- [案例 3：项目审计与整改任务](case-03-project-remediation.md)
- [before / after 对照](before-after-comparison.md)

## 边界说明

- 原始运行样本主要保存在本地 `.local-docs/`
- 这些案例经过了整理和重写
- 它们不等于默认 runtime memory

资产边界详见 [../runtime-asset-boundary.md](../runtime-asset-boundary.md)。
