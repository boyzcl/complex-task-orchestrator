# Definition Of Done

用本参考判断当前复杂任务是否真的完成。

## 主任务完成

当以下条件都满足时，可以说主任务已完成：

- 需求已经确认
- 测试与验收方案已经生成
- 执行计划已经生成并落盘
- 核心交付物已经完成
- 已按预设验证方案回验
- 必过项已通过，或外部阻塞已说明
- 已进行了自检
- 已给出最终总结

## 主任务未完成

出现以下情况之一时，不能说主任务完成：

- 需求还没确认
- 测试与验收方案还没生成
- 执行计划还没生成
- 核心交付物还没产出
- 必过验证项失败且尚未迭代或说明阻塞
- 没有按预设验证方案回验
- 自检还没做
- 关键阻塞尚未说明

## 扩展验证

以下内容默认属于扩展验证，而不是主流程未完成：

- forward test
- 真实场景压力测试
- 泛化任务测试
- 第二个任务上的复用验证

注意：预设验证回验不是扩展验证。它是复杂或长链路任务主流程的一部分。

如果要做扩展验证，请明确说明：

- 主任务已完成
- 当前进入的是扩展验证阶段
- 扩展验证的目标是什么

## 最终表述规范

优先使用以下表达：

- `主任务已完成，下面进入扩展验证。`
- `当前交付已经完成，预设验证和自检已完成；剩余的是可选的 forward test。`
- `主流程结束，下面验证它在其他任务上的泛化能力。`
